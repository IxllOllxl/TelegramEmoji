# TelegramEmoji
Telegram Emojis
Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}Copy
Step 2. Add the dependency

	dependencies {
	        compile 'com.github.kaitkulov:TelegramEmoji:0.1.2'
	}
